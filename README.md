# Coursera-End-of-Course-Portfolio-Waze-Project-

Waze User Churn Prediction — Course 1 Portfolio Project

# Project Overview

This repository contains my Course 1 portfolio project for the Google Advanced Data Analytics Certificate. The project simulates a real‑world analytics workflow at Waze, where the objective is to begin developing a machine learning solution to predict monthly user churn.

This phase of the project focuses on foundational data work:

Constructing a clean, well‑structured DataFrame

Inspecting and validating data types

Generating descriptive statistics

Preparing the dataset for exploratory data analysis (EDA)

Communicating early findings to technical and non‑technical stakeholders

These steps establish the groundwork for later modeling and hypothesis testing.

Technical Focus
This project demonstrates core competencies expected in an entry‑level data analyst or junior data scientist role:

Data Engineering & Preparation
Loading and structuring raw data into a Pandas DataFrame

Verifying schema integrity and identifying data type inconsistencies

Assessing data completeness and potential quality issues

Exploratory Data Foundations
Producing descriptive statistics for numerical and categorical variables

Identifying early trends, anomalies, and potential predictors of churn

Documenting considerations for downstream EDA and modeling

Communication & Documentation
Translating technical observations into concise summaries

Tailoring communication for both technical and cross‑functional audiences

Applying the PACE workflow for structured project planning
