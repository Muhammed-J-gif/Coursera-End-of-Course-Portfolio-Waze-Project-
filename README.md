# Waze User Churn Prediction — Course 1 Portfolio Project  
*Google Advanced Data Analytics Certificate*

## Project Overview  
This repository contains my Course 1 portfolio project for the Google Advanced Data Analytics Certificate. The project simulates an analytics workflow at Waze, where the objective is to begin developing a machine learning solution to predict monthly user churn.

This phase focuses on foundational data work essential for any predictive modeling pipeline:

- Constructing a clean, well‑structured DataFrame  
- Inspecting and validating data types  
- Generating descriptive statistics  
- Preparing the dataset for exploratory data analysis (EDA)  
- Communicating early findings to both technical and non‑technical stakeholders  

---

This project demonstrates core competencies relevant to data analyst and junior data scientist roles.

- Generated descriptive statistics for numerical and categorical variables
